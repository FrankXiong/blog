Fun言

2015.7
Java

Fun言是一款粤语学习应用，主要功能有日常对话、听书、粤语歌曲、教学视频，通过文字、语音、视频等多种方式帮助用户轻松快速地学习粤语。同时，它还具备语音合成和语音听写的功能，可自动将用户输入的文字转化为各地方言语音，也可接受来自用户的语音，并将其翻译成文字。本应用集成了社交分享的功能，在“方言角”里，用户可以添加好友，发布动态，以及点赞等。
